# Filebrowser Quantum

A lightweight Docker Compose setup for running the **Filebrowser Quantum** container.
The future of Filebrowser was in question so switched to Quantum.

## Quick Notes

Proxy Port: 80

Default user account:
* Username: admin
* Password: admin
